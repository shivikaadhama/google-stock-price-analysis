# Google Stock Price Analysis

## Project Overview
This project analyzes the historical stock performance of Google (Alphabet Inc.) from 2004 to 2023.  
The objective is to explore price trends, volatility, trading volume, and technical indicators using Python.

## Dataset
The dataset contains daily stock information including:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Adjusted Close
- Trading Volume

Total records: **4842 trading days**

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
1. Data Cleaning and Preprocessing
2. Stock Price Trend Visualization
3. Moving Average Analysis (50-day & 200-day)
4. Volatility Analysis
5. Trading Volume Analysis
6. Correlation Heatmap

## Key Insights
- Google stock shows strong long-term growth since its IPO.
- Moving averages confirm a sustained bullish trend.
- Volatility increased in recent years due to market fluctuations.
- Trading volume was highest during early years after IPO.

## Conclusion
This analysis demonstrates how historical stock data can be explored using Python to identify trends, volatility patterns, and relationships between financial indicators.
